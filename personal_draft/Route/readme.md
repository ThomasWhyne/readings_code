  - 导航时无须重新加载页面是 `SPA` 的关键特性，赋予用户原生应用般的体验。    
  - `SPA` 通常会在设计中融入 `Routing`:  
 借助MV*框架或第三方库代码的实现，将URL风格的路径与功能关联起来。  
 路径通常看起来像相对URL，其充当用户导航时到达特定视图的触发因素。  
 路由器可以动态更新浏览器的URL，并允许用户使用前进、后退按钮，强化屏幕某部分变化是会到达新位置的理念。
